# digest

Пользователь запрашивает **внеочередной дайджест** новостей прямо сейчас.

## Инструкции:

1. Собери 10-15 свежих новостей по приоритетным темам (ИИ, Вайбкодинг, Робототехника, eVTOL, Технологии, Бизнес, Инвестиции)
2. Используй веб-поиск для поиска актуальных новостей
3. Отправь дайджест СТРОГО по формату из `digest-format-final.md`
4. Двухшаговая отправка: текст → editMessage с кнопками
5. Минимум 30% новостей из новых источников
6. Учитывай систему рейтинга из `feedback-learning-system.md`

## Если пользователь указал тему:
- Сфокусируй дайджест на указанной теме
- Например: `/digest ИИ` — только новости про ИИ

## Формат ответа:
Используй формат из `digest-format-final.md`. Не отправляй итоговое сообщение.
