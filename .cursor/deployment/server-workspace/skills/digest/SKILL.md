# digest

> **ОБНОВЛЕНИЕ:** Для глубокого анализа конкретных URL используй `/smart-digest` —
> он открывает страницу в браузере, делает скриншот, извлекает контент и анализирует
> позицию автора. Используй `/digest` для автоматического сбора новостей по ключевым словам,
> `/smart-digest` — когда есть конкретные URL для анализа.

Пользователь запрашивает **внеочередной дайджест** новостей прямо сейчас.

## Инструкции:

1. **Прочитай** `data/sent-digests.json` для дедупликации
2. Собери 10-15 **свежих** новостей (только за последние 48 часов!) по приоритетным темам (ИИ, Вайбкодинг, Робототехника, eVTOL, Технологии, Бизнес, Инвестиции)
3. Используй веб-поиск для поиска актуальных новостей
4. Отправь дайджест СТРОГО по формату из `digest-format-final.md`
5. **НЕ добавляй inline кнопки** — оценка через нативные реакции Telegram
6. Минимум 30% новостей из новых источников
7. **После отправки** запиши отправленные новости в `data/sent-digests.json`
8. **После каждого sendMessage** сохрани message_id в `data/dual-rating-data.json` → `messageHistory` (подробный алгоритм в `digest-format-final.md` → секция "Сохранение message_id")

## Если пользователь указал тему:

- Сфокусируй дайджест на указанной теме
- Например: `/digest ИИ` — только новости про ИИ

## Если пользователь указал конкретные URL:

- Перенаправь на `/smart-digest` — он сделает глубокий анализ через браузер

## Формат ответа:

Используй формат из `digest-format-final.md`. Не отправляй итоговое сообщение.
